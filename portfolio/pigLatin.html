Pig Latin Translator

jQuery, BootStrap, CSS3, HTML5

CHALLENGE

This project started as a challenge to translate one word. As the project grew, the translateButtonClicked() function became more complex. This led to a rather long, convoluted function.

SOLUTION

Refactoring the action of the translateButtonClicked() function allowed each substep in the text translation process to be wrapped in its own function. This created the helper functions punctuationHandler(), capitalization(), and joinWord(). Each of these subfunctions was easier to test and modify than the previous, large translateButtonClicked() function.
